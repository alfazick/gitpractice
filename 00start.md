
## Rapid Intro

Git Workflow

1) Changes to files in working directory
   ↓
Get added to / Get unstaged from
   ↓
2) Staging area / Added content / Git index
   ↓
Are committed to
   ↓
3) Local Git repository
   ↓
Are pushed to / Are fetched from
   ↓
4) Remote Git repository
